## IT Salary Analysis

## Project Overview

This project analyzes salary data in the IT industry across different countries, roles, and experience levels.

The goal is to understand salary distribution and identify factors that influence income.

---

## Dataset

The dataset was synthetically generated and includes:

- Country — employee location  
- Position — job role  
- Salary — monthly salary  
- Experience — Junior / Middle / Senior  
- Remote — remote or office work  

---

## Tools Used

- Python
- pandas
- numpy
- matplotlib

---

## Key Steps

### 1. Data Preparation

- Generated synthetic dataset
- Created annual salary feature:
  - **Salary_usd_year = Salary × 12**

---

### 2. Analysis

- Average salary by position
- Average salary by country
- Salary by experience level
- Remote vs office salary comparison

---

### 3. Advanced Analysis

- Salary distribution using boxplot
- Senior salary analysis by country
- Data Scientist salary comparison across countries

---

## Visualizations

- Bar charts for salary comparisons
- Boxplot for salary distribution
- Comparative analysis for senior roles

---

## Key Insights

- Salary strongly depends on experience level.
- Country significantly impacts salary levels.
- Senior roles show the highest salary variation.
- Remote work has almost no impact on salary levels.
