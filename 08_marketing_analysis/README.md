## Marketing Campaign Analysis

## Project Overview

This project analyzes marketing campaign performance across different channels and countries.

The goal is to evaluate campaign efficiency and identify the most profitable channels using marketing metrics.

---

## Dataset

The dataset was synthetically generated and includes:

- campaign — campaign name  
- channel — marketing channel  
- impressions — number of views  
- clicks — number of clicks  
- conversions — number of purchases  
- cost — campaign cost  
- country — target country  

---

## Tools Used

- Python
- pandas
- numpy
- matplotlib
- seaborn

---

## Key Metrics

- CTR (Click-Through Rate) = clicks / impressions  
- CR (Conversion Rate) = conversions / clicks  
- CPA (Cost per Acquisition) = cost / conversions  
- Revenue = conversions × average order value  
- ROI (Return on Investment)

## Key Insights

- Different channels have varying efficiency and profitability.
- High traffic (CTR) does not always lead to high revenue.
- ROI is the most important metric for evaluating marketing performance.
- Some channels are cost-efficient (low CPA), while others are more profitable (high ROI).
